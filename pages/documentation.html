Documentation

Installation

Get started with SoLink AI by installing our npm package:

Terminal 
npm install solink-ai

Or if you're using yarn:

Terminal 
yarn add solink-ai

Quick Start

Initialize SoLink AI in your project:

JavaScript 
import { SoLinkAI } from 'solink-ai';

const solink = new SoLinkAI({
    apiKey: 'your-api-key',
    network: 'mainnet',
    config: {
        aiModel: 'gpt-4',
        maxRetries: 3
    }
});

// Initialize tracking
await solink.init();

Configuration

Configure your SoLink AI instance with these options:

Option Type Default Description
apiKey string required Your SoLink AI API key
network string 'mainnet' Solana network to connect to
aiModel string 'gpt-4' AI model for analytics

AI Analytics

Implement AI-powered analytics in your application:

JavaScript 
// Track wallet activity
const analysis = await solink.analyze({
    wallet: '0x742d35cc6634c0532925a3b8448c454e4438f44e',
    timeframe: '24h',
    metrics: ['transactions', 'volume', 'patterns']
});

// Get AI insights
const insights = await solink.getInsights(analysis.id);

console.log(insights.summary);
console.log(insights.recommendations);
